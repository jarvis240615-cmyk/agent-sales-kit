# LinkedIn Connection Request Templates

All under 300 characters. Pick the one that fits.

---

**1. Direct value**
```
Hi {{first_name}}, I help [target companies] with [pain point]. Saw {{company}} is [observation] — thought it'd be worth connecting.
```

**2. Content engagement**
```
Hi {{first_name}}, your post on [topic] resonated. I work with [similar companies] on [problem]. Would love to connect.
```

**3. Company news**
```
Hi {{first_name}}, congrats on [news]. I work with companies at this stage on [service]. Worth connecting?
```

**4. Question**
```
Hi {{first_name}}, is [pain point] something {{company}} is working on? I might have something useful — connecting to share.
```

**5. Mutual**
```
Hi {{first_name}}, [Name] suggested we connect. I work with [companies like yours] on [topic]. Happy to connect.
```
